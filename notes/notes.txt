- Все в спринге вертиться вококруг spring core.
- Dependency Injection фреймворк определяет и внедряет зависимости объекта через:
  • параметры конструктора
  • параметры статического метода инициализации (фабричный метод)
  • свойства объекта (set* methods)
- loC Container - это объект, который занимается созданием других объектов и    внедрением в них зависимостей (DI).
  Bean - это объект со всеми необходимыми зависимостями, который был создан с помощью loC Container (Controller, Service, Repository)

  Metadata - предствляет собой совокупность объектов (Bean Definitions), которые говорят loC Container как создавать Beans, конфигурировать и внедрять их зависимости

  Plain Old Java Object - обычный объект, с набором сарйств, конструкторов и get/set методов. Не содержит логики (DTO, Entity)

  3 пособа как создавать bean:
  XML-based, Annotation-based, Java-based
